# Подключение MAX9814 к Luckfox Pico Ultra W

## Что уже есть внутри модуля TZT MAX9814

Схема модуля (по даташиту и маркировке на плате):
- **Микрофон** — встроен на модуль
- **C_IN** (DC-блок на входе микрофона) — встроен, снаружи не нужен
- **R_MICBIAS** (смещение для микрофона) — встроен
- **C_OUT** (DC-блок на выходе) — **НЕ встроен в TZT модуль**

Вывод: пин OUT модуля имеет постоянное DC-смещение **1.25V** (указано
на плате). Конденсатор C1 (DC-блок) снаружи **обязателен**.

---

## Компоненты (обвязка, нужно собрать снаружи)

| Обозначение | Номинал                      | Назначение                                              |
|-------------|------------------------------|---------------------------------------------------------|
| C1          | **любой 100 nF – 10 µF**     | DC-блок: убирает 1.25V DC с пина OUT модуля            |
| R1          | 100 kΩ                       | Подтяжка MIC1P к MICBIAS (псевдо-дифф. баланс)        |
| R2          | 100 kΩ                       | Подтяжка MIC1N к MICBIAS (псевдо-дифф. баланс)        |
| C2          | 100 nF, керамика             | AC-шунт MIC1N на GND (подавление ВЧ-шума)             |

### Чем заменить C1 если нет 1 µF

Нижняя граница полосы: $f_{-3dB} = \frac{1}{2\pi \cdot C1 \cdot R1}$

| C1        | f_-3dB при R1=100kΩ | Вывод                          |
|-----------|---------------------|--------------------------------|
| 10 µF     | ~0.16 Hz            | Идеально                       |
| 1 µF      | ~1.6 Hz             | Отлично                        |
| **100 nF**| **~16 Hz**          | **Подходит** (ниже голоса)     |
| 47 nF     | ~34 Hz              | Приемлемо, небольшой срез баса |

Если есть только **100 nF** (например, как C2) — использовать его как C1,
всё будет работать нормально. Если есть **электролит 10 µF** — тоже
подходит: плюсом к MAX9814 OUT (сторона с 1.25V DC), минусом к R1/MIC1P.

## Схема подключения (ASCII)

```
MAX9814 module          Обвязка                  Luckfox Pico Ultra W
──────────────          ──────────               ──────────────────────

VDD ─────────────────────────────────────────────  3V3   (НЕ 1.8V — минимум 2.7V)
GND ─────────────────────────────────────────────  GND
AR  ─── не подключать

                                    ┌────────────  MICBIAS
                                    │
                               R1 [100kΩ]
                                    │
OUT ───── C1 ──────────────── [точка A] ─────────  MIC1P
        (+ к OUT,                   │
         - к точке A)               │
                               R2 [100kΩ]
                                    │
                               [точка B] ──────────  MIC1N
                               MICBIAS ──── R2 ─── точка B   (MICBIAS к обоим через резисторы)
                                    │
                               C2 [100nF]
                                    │
                                   GND

GAIN ────────────────────────────────────────────  не подключать  (50 dB, рекомендуется)
```

## Таблица соединений — строка за строкой

| № | Откуда               | Через что             | Куда                  |
|---|----------------------|-----------------------|-----------------------|
| 1 | MAX9814: **VDD**     | прямой провод         | Luckfox: **3V3**      |
| 2 | MAX9814: **GND**     | прямой провод         | Luckfox: **GND**      |
| 3 | MAX9814: **OUT**     | C1 (+ вывод к OUT)    | C1 (- вывод = точка A)|
| 4 | Точка A              | прямой провод         | Luckfox: **MIC1P**    |
| 5 | Точка A              | R1 (100kΩ)            | Luckfox: **MICBIAS**  |
| 6 | Luckfox: **MICBIAS** | R2 (100kΩ)            | Точка B               |
| 7 | Точка B              | прямой провод         | Luckfox: **MIC1N**    |
| 8 | Точка B              | C2 (100nF)            | Luckfox: **GND**      |
| 9 | MAX9814: **AR**      | не подключать         | —                     |
|10 | MAX9814: **GAIN**    | не подключать (50 dB) | —                     |

**Точка A** = узел: [C1 минус] + [R1 один конец] + [MIC1P]  
**Точка B** = узел: [R2 один конец] + [MIC1N] + [C2 плюс]  
**MICBIAS** = узел: [R1 другой конец] + [R2 другой конец]

## Принципиальная схема

```
3V3 ──────────────────────────────────────────── VDD (MAX9814)
GND ──────────────────────────────────────────── GND (MAX9814)

                                  MICBIAS (Luckfox)
                                       │
                              ┌────────┴────────┐
                           R1 │              R2 │
                          100kΩ             100kΩ
                              │                 │
MAX9814 OUT ──[C1]──────── MIC1P            MIC1N ──[C2 100nF]── GND
                (+→OUT)  (Luckfox)        (Luckfox)
                (-→MIC1P)
```

## Пояснения

### Почему VDD → 3V3, а не 1.8V?
MAX9814 требует минимум 2.7V. Пины MIC1P/MIC1N/MICBIAS — домен 1.8V кодека
RV1106. C1 изолирует оба домена по постоянному току (DC), пропуская только
звуковой сигнал (AC).

### Полярность C1 (если электролит)
Плюс (+) → сторона MAX9814 OUT (там 1.25V DC).  
Минус (−) → сторона MIC1P / точка A (там ~0.9V от MICBIAS через R1).

### Зачем MICBIAS + R1 + R2?
RV1106 ACODEC — дифференциальный вход. Оба входа (MIC1P и MIC1N) должны
иметь одинаковое DC-смещение относительно аналоговой земли кодека.
MICBIAS ≈ 0.9V — это "виртуальная земля" 1.8V-домена. R1 и R2 по 100kΩ
создают это смещение, не нагружая MAX9814 и не разряжая MICBIAS.

### Зачем C2?
MIC1N — пассивный инвертирующий вход. C2 шунтирует ВЧ-наводки на нём,
повышая CMRR (подавление синфазных помех).

### Настройка GAIN (исправлено по даташиту)
| GAIN пин          | Усиление | Рекомендация                        |
|-------------------|----------|-------------------------------------|
| **Не подключать** | **50 dB**| **Рекомендуется для старта**        |
| → GND             | 40 dB    | Для громкого/близкого источника     |
| → VDD (3V3)       | 60 dB    | Для тихого помещения, далёкий голос |

## Частота дискретизации

По умолчанию используется 16000 Hz (wideband, полоса 300–7000 Hz).
Для экономии трафика можно переключить на 8000 Hz (narrowband, 300–3400 Hz):

```bash
./build.sh --sensor MIS5001 --audio on --sample-rate 8000
```
